# Organização de Atendimentos

Sistema desenvolvido em Python para organização automática de atendimentos médicos em consultórios.

O sistema distribui os atendimentos em consultórios sem conflito de horários, priorizando atendimentos expressos e reutilizando consultórios já existentes sempre que possível.

## Funcionalidades

* Cadastro de pacientes
* Organização automática de consultórios
* Priorização de atendimentos expressos
* Verificação de conflitos de horário
* Interface gráfica utilizando Tkinter
* Estrutura orientada a objetos

## Tecnologias utilizadas

* Python
* Tkinter

## Estrutura do projeto

```bash
src/
│
├── app.py
├── atendimento.py
├── consultorio.py
└── organizador.py
```

## Como executar

Entre na pasta src:

```bash
cd src
```

Execute o sistema:

```bash
python app.py
```
